INSTRUCCIONES DE DESPLIEGUE Y DESARROLLO

1. DESPLIEGUE AUTOMATIZADO (Recomendado para demo)
Asegúrate de tener Docker Desktop iniciado. Simplemente ejecuta en la raíz:
  docker compose up --build

2. DESARROLLO LOCAL (Si necesitas modificar el código fuente)
Si requieres ejecutar los servicios fuera de contenedores para debugging:

  A) Servicio de Autenticación (Python):
     - Navega a ./PythonAuthentication
     - Crea entorno: python -m venv venv
     - Activa: .\venv\Scripts\activate (o source venv/bin/activate en Mac)
     - Instala: pip install -r requirements.txt
     - Ejecuta: uvicorn main:app --reload

  B) Servicio de Chat (Go):
     - Requiere Go 1.26+ instalado.
     - Ejecuta desde la raíz: go run ./cmd/server/main.go
